**Money-Transfer using Quarkus**

Provides 3 REST endpoints -
<br>
1. /register-account
2. /check-balance
3. /transfer-money

Build command to build the project -
<br>
`./mvnw clean install`

To run the project -
<br>
`java -jar target/money-transfer-1.0.0-runner.jar`

Access the swagger-ui endpoint -
<br>
`http://localhost:8080/swagger-ui`
